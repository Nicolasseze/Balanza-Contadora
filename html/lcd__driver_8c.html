Detailed Description

Implementación del controlador de alto nivel para un LCD tipo HD44780 vía I2C.

Este archivo contiene las funciones necesarias para inicializar el display, enviar comandos, imprimir texto y posicionar el cursor. Utiliza la capa lcd_port para acceder al hardware físico mediante I2C.
