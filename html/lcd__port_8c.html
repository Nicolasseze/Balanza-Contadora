Detailed Description

Implementación de funciones de bajo nivel para manejar un LCD 20x4 mediante I2C usando PCF8574.

Este archivo contiene la lógica de inicialización del LCD y el envío de datos o comandos a través de una interfaz I2C, utilizando el periférico configurado con HAL.

Function Documentation

◆ lcdPortInit()

void lcdPortInit ( void  )

Inicializa el LCD en modo de operación de 4 bits.

Inicializa el puerto I2C y configura el entorno necesario para el LCD.

Debe ser llamada una vez tras el encendido del sistema. Ejecuta la secuencia de inicialización especial exigida por la hoja de datos del controlador del LCD.
