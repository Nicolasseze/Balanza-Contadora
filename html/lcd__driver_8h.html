Detailed Description

Interfaz de alto nivel para el control de un LCD mediante comandos compatibles con HD44780.

Este módulo encapsula el control lógico del LCD, permitiendo enviar comandos, posicionar el cursor y mostrar texto, todo sobre una interfaz abstracta independiente del puerto físico.
