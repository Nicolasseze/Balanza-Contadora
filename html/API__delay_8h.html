Detailed Description

Módulo para manejo de retardos no bloqueantes.

Author
Nicolás
Date
2025

Este módulo permite la gestión de retardos mediante el uso de estructuras, evitando el uso de delay() y mejorando el rendimiento en sistemas embebidos.
