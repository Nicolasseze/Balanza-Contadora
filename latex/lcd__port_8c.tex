\doxysubsection{Detailed Description}
Implementación de funciones de bajo nivel para manejar un LCD 20x4 mediante I2C usando PCF8574. 

Este archivo contiene la lógica de inicialización del LCD y el envío de datos o comandos a través de una interfaz I2C, utilizando el periférico configurado con HAL.
